# 첨단 기술에 오신 것을 환영합니다!

Isaac Lab은 오픈소스로 공개되어, 로봇 시뮬레이션 분야에서의 개방형 협력 커뮤니티를 육성하는 것이 우리의 의도입니다.
우리는 강력한 도구가 로봇공학의 미래에 있어 필수적이라고 믿습니다.

때때로 새로운 기능은 Isaac Lab의 내부 구조에 광범위한 변경을 요구할 수 있습니다.
기능이 완성되지 않은 상태에서 전체 커뮤니티의 피드백 없이 직접 통합하면, 준비되지 않은 사용자에게 심각한 문제를 초래할 수 있습니다.

이를 해결하기 위해, 일부 주요 기능은 **실험적 기능 브랜치**(Experimental Feature Branches)로 먼저 출시됩니다.
이렇게 함으로써 커뮤니티는 해당 기능이 완전히 통합되기 전에 실험하고 기여할 수 있으며, 예기치 못한 새로운 오류로 인해 개발이 좌초될 가능성을 줄일 수 있습니다.

# 센서

이 섹션에서는 Isaac Lab에서 제공하는 다양한 센서 API를 개괄합니다.

Isaac Lab의 모든 센서는 씬의 “측정값”에 대한 접근을 제공하는 핵심 기능을 갖는 `SensorBase` 추상 클래스를 상속받습니다. 이러한 측정값은 레이캐스팅 결과, 카메라 렌더링 이미지, 또는 시뮬레이션에서 직접 쿼리된 지상 진실 데이터(예: 포즈)와 같이 다양한 형태를 가질 수 있습니다. 데이터의 형태에 관계없이, 센서는 씬을 쿼리하여 주기적으로 측정값으로 버퍼를 업데이트하는 버퍼를 갖는다고 생각할 수 있습니다. 이 `update_period`는 “시뮬레이션된” 초 단위로 정의되므로, 시뮬레이션의 시간 흐름이 실제 세계에 비해 확장되어도 센서는 적절한 비율로 업데이트됩니다. `SensorBase`는 벡터화 가능성을 염두에 두고 설계되었으며, 클론된 환경 전체에 걸친 센서 복사본들의 버퍼를 보유합니다.

버퍼 업데이트는 `SensorBase` 클래스의 `_update_buffers_impl` 추상 메서드를 오버라이드하여 수행됩니다. 시뮬레이션의 모든 타임스텝 `dt`에서, 모든 센서가 업데이트를 위해 쿼리됩니다. 이 쿼리 동안, 해당 센서가 관리하는 모든 버퍼에 대해 마지막 업데이트로부터 경과한 시간이 `dt`만큼 증가합니다. 경과한 총 시간이 버퍼의 `update_period`보다 크거나 같으면, 해당 버퍼가 다음 쿼리에서 업데이트될 플래그가 설정됩니다.
